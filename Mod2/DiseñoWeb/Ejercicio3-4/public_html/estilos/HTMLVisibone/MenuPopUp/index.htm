VisiBone Popups 1.05

The HTML Popup is a tall skinny on-screen quick reference showing all HTML tags and attributes in use or universal. It includes color-coded indications of support by Netscape®, Microsoft® Internet Explorer and W3C®, plus style sheet alternatives, and bug and clash icons with details online. It is taken from a page of the VisiBone HTML Card (September 2000 edition).

The Color Popups show all 216 web-safe colors in a wheel, the same arrangement as the VisiBone2 swatch libraries shipped in Adobe® Photoshop® and Illustrator®, BBEdit® and The Gimp. The same arrangement is used in the VisiBone Color Card.


You're welcome to:
  • Popup now:   (Version 4 browsers may show broken images: try desktop install instead.)
  • Desktop install:
    1. Unzip visbopop.zip to a directory on your hard disk or network (using PKZip, WinZip or ZipIt).
    2. Windows: From that directory, right-click-drag deskpop.htm to your desktop, choose "Create Shortcut Here". Rename (F2) to "VisiBone Popups". Tip (thanks to John Herring): uncheck the IE advanced property "Reuse windows for launching shortcuts". When checked, the popups commandeer existing browser windows.
      Mac: From that directory, Hold down OPT + CMD keys and drag deskpop.htm to your desktop. Or select the file and press CMD-L (File / Make Alias menu). It's CMD-M for OS9. (((Is this accurate & complete? Feedback welcome.)))
  • Mirror on your web site:
    1. Unzip visbopop.zip to your web server (using PKZip, WinZip or ZipIt).
    2. Link to index.htm (or modify or rewrite).
